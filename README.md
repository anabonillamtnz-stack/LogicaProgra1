## Number Comparison Tool (Java)
A professional Java console application designed to process and analyze numerical inputs. This project demonstrates core programming logic, including variable management, conditional structures, and sorting algorithms.

### Key Features
* **User Input Handling: Prompts the user to enter three numbers and stores them in their respective variables.**

* **Data Analysis: Automatically identifies the largest, middle, and smallest numbers among the inputs.**

* **Sorted Output: Prints the numbers to the Standard Output (Console) in two formats:**

   * **Sorted from largest to smallest.**

   * **Sorted from smallest to largest.**

* **Equality Check: Determines if the numbers are equal and provides a specific message if a match is found.

### Technical Validation (Test Cases)
The program has been rigorously tested with the following scenarios to ensure logical consistency:

### Scenario 1
* **Inputs:** `4, 4, 2`
* **Outputs:**
  * **Largest to Smallest:** `4, 4, 2`
  * **Smallest to Largest:** `2, 4, 4`

### Scenario 2
* **Inputs:** `4, 2, 4`
* **Outputs:**
  * **Largest to Smallest:** `4, 4, 2`
  * **Smallest to Largest:** `2, 4, 4`
 
## Temperature Converter - Logic Challenge 02

This project is a web-based temperature converter built with JavaScript, HTML, and CSS. It allows users to convert between Celsius, Fahrenheit, and Kelvin through an interactive menu.

### Description

The program follows a logical series of steps to solve specific conversion problems. It captures user input via the DOM (HTML inputs) and interactive prompts, performs mathematical calculations using custom functions, and validates that the input data is of a numerical type.

### Features

Six Conversion Options:Celsius to Fahrenheit
* Fahrenheit to Celsius
* Celsius to Kelvin
* Kelvin to Celsius
* Fahrenheit to Kelvin
* Kelvin to Fahrenheit

* Data Validation: Identifies if the input is a number. If not, it displays an error message and prevents the calculation.
* Clean Logic: Uses modular functions for each mathematical conversion to ensure code reusability and clarity.

### Technologies UsedJavaScript (ES6+): 
* Logic, DOM manipulation, and mathematical functions.
* HTML5: Structure and user input fields.
