## Number Comparison Tool (Java)
A professional Java console application designed to process and analyze numerical inputs. This project demonstrates core programming logic, including variable management, conditional structures, and sorting algorithms.

## Key Features
* **User Input Handling: Prompts the user to enter three numbers and stores them in their respective variables.**

* **Data Analysis: Automatically identifies the largest, middle, and smallest numbers among the inputs.**

* **Sorted Output: Prints the numbers to the Standard Output (Console) in two formats:**

   * **Sorted from largest to smallest.**

   * **Sorted from smallest to largest.**

* **Equality Check: Determines if the numbers are equal and provides a specific message if a match is found.

## Technical Validation (Test Cases)
The program has been rigorously tested with the following scenarios to ensure logical consistency:

### Scenario 1
* **Inputs:** `4, 4, 2`
* **Outputs:**
  * **Largest to Smallest:** `4, 4, 2`
  * **Smallest to Largest:** `2, 4, 4`

### Scenario 2
* **Inputs:** `4, 2, 4`
* **Outputs:**
  * **Largest to Smallest:** `4, 4, 2`
  * **Smallest to Largest:** `2, 4, 4`
